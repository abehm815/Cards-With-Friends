# Cards With Friends

## Overview
Cards With Friends is a multiplayer Android card-game app developed as a team project for COMS 3090 at Iowa State University. The app provides a social gaming experience with real-time lobby matchmaking, in-lobby chat, and competitive stats across multiple classic card games.

Demo Video: https://www.youtube.com/watch?v=ZCWio5p7RMI&list=PL6BdlkdKLEB-RhBjBTGRFWf87xXYjf9tq&index=23

## Core Features

### Authentication & Profiles
- User sign-up, login, and profile management
- Persistent player statistics
- Logout and stat reset options

### Multiplayer Lobby System
- Create a lobby with a join code
- Join or discover active lobbies in real time
- In-lobby player list with host controls

### Real-Time Chat
- WebSocket-based in-lobby chat
- Unread message tracking

### Navigation & UI
- Central home hub for game selection
- Persistent bottom navigation for stats, leaderboards, and profile

### Competitive Tracking
- Per-game statistics dashboard
- Win-percentage leaderboards
- Match history with detailed views

## Games Included

### Blackjack
- Real-time multiplayer gameplay
- Betting system with chip tracking
- Dealer and player action handling

### Euchre
- Trick-taking gameplay with teams
- Bidding and trump selection
- Team score tracking

### Go Fish
- Host-controlled game start
- Player ask actions
- Live game state updates

### Crazy 8s
- Color selection mechanics
- Draw penalties
- Live deck and turn tracking

## Project Context
This project was developed for COMS 3090 at Iowa State University and focuses on real-time multiplayer systems, mobile UI design, and game logic integration within a single Android application.
